# vaccination.py

Get COVID-19 vaccination schedules from booking.moh.gov.ge in the CLI.

## Installation
```bash
$ pip install vaccination
```

## Usage

Make sure the Python bin directory is added to your `PATH`

```bash
$ vaccination
```
